Rental Kompanion API

Welcome to the Rental Kompanion API. Here are the available endpoints:

GET /api/health

Check API health status

POST /api/auth/register

Register a new user

POST /api/auth/login

Login user

GET /api/auth/me

Get current user profile

POST /api/auth/logout

Logout user

GET /api/companions

Get list of available companions

GET /api/companions/:id

Get companion details
